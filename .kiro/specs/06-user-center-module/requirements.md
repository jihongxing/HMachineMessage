# 用户中心模块 - 需求文档

## 功能范围

### 1. 个人信息
- 查看个人资料
- 修改昵称/头像
- 修改密码
- 更换手机号

### 2. 认证管理
- 实名认证（姓名+身份证）
- 企业认证（营业执照+法人信息）
- 认证状态查看

### 3. 我的设备
- 已发布/审核中/已拒绝/已下架
- 设备数据统计
- 快捷操作（编辑/删除/上下架）

### 4. 我的收藏
- 收藏列表
- 取消收藏
- 设备状态显示

### 5. 浏览历史
- 最近浏览30条
- 清空历史

### 6. 我的订单
- 订单列表（待支付/已支付/已退款）
- 订单详情
- 发票申请

### 7. 账户管理
- 余额显示
- 充值入口
- 消费记录
- 提现功能（认证用户）

### 8. 消息中心
- 通知列表
- 未读数量
- 消息筛选

## API接口

### 个人信息
1. GET /api/v1/user/profile - 获取资料
2. PUT /api/v1/user/profile - 更新资料
3. PUT /api/v1/user/password - 修改密码
4. PUT /api/v1/user/phone - 更换手机号

### 认证
5. POST /api/v1/user/verify/realname - 实名认证
6. POST /api/v1/user/verify/company - 企业认证

### 账户
7. GET /api/v1/user/balance - 余额查询
8. GET /api/v1/user/transactions - 消费记录

## 验收标准

- 信息修改实时生效
- 认证流程完整
- 数据统计准确
- 页面响应流畅
