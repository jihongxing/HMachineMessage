6.1 **审核机制**：

6.2 **审核标准**：

7.2 **普通用户**（1级）：

7.3 **优质用户**（2级）：

7.4 **认证用户**（3级）：

8.2 **中度违规**（虚假信息、重复发布）：

- 强制人工审核

8.3 **严重违规**（违法内容、恶意刷单）：
