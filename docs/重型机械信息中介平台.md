# PRD 文档：重型机械信息中介平台（免费基础 + 月度增值排名付费）

## 一、项目概述

**项目名称**：重型机械信息中介平台

**项目定位**：

- 为中国农业、工程及其他重型机械行业提供全国性 **信息展示和撮合服务**
    
- 平台仅提供信息展示和供需撮合，不参与作业调度或交易结算
    
- 核心目标：**解决重型机械供需不匹配、信息不对等问题**
    

**核心理念**：

> “免费基础展示 + 增值排名付费 → 高市占率 → 高机械利用率 → 全国覆盖”

---

## 二、目标用户

1. **需求方**
    
    - 农户、小型农场、农业合作社
        
    - 建筑/施工承包商、租赁公司
        
    - 使用大型机械完成农业作业或工程施工
        
2. **供给方**
    
    - 农机手、农机合作社
        
    - 工程机械租赁公司、个人设备拥有者
        
    - 提供机械服务或租赁，希望获得曝光

## 三、核心功能需求

### 3.1 核心信息展示（免费基础）

- 机械类型、型号、作业能力
    
- 可租日期/作业时间
    
- 所在区域/地理位置
    
- 联系方式（电话、微信、企业联系方式）
    
    - **隐私保护机制**：联系方式默认不直接展示
        
    - 用户需点击"查看手机号"或"查看微信"按钮
        
    - 前端发送请求到后台获取联系方式
        
    - 后台记录查看行为（用于数据统计和防刷）
        
    - 返回联系方式后前端展示
        
- 用户评价（可选）
    

### 3.2 搜索与筛选

- 按 **一级分类 / 二级分类 / 地区 / 作业时间 / 推荐位** 筛选
    
- 排序：距离、热度、推荐位
    
- 地图展示（可选）
    

### 3.3 后台管理

**智能审核系统（减少人工审核强度）**

1. **自动审核机制**
   
   - **内容合规检测**
     - 文本敏感词过滤（政治、色情、违法内容）
     - 图片AI识别（OCR文字检测、色情暴力识别）
     - 联系方式格式验证（手机号、微信号合法性）
   
   - **信息完整性校验**
     - 必填字段检测（型号、地区、价格、图片）
     - 图片数量要求（≥3张）
     - 描述字数要求（≥50字）
   
   - **异常行为识别**
     - 同一用户短时间大量发布（自动限流）
     - 重复内容检测（自动去重）
     - 价格异常检测（过高/过低自动预警）

2. **用户信用体系**
   
   - **用户分级管理**
     - 新用户：首次发布需人工审核
     - 普通用户（5条通过）：自动审核 + 5%抽查
     - 优质用户（20条通过 + 0违规）：免审核直接发布
     - 认证用户（企业认证）：免审核 + 优先展示
   
   - **违规惩罚机制**
     - 轻微违规（信息不完整）：警告 + 要求修改
     - 中度违规（虚假信息）：降级 + 强制人工审核
     - 严重违规（违法内容）：封号 + 黑名单

3. **用户自治机制**
   
   - **举报系统**
     - 用户举报按钮（虚假信息、违规内容）
     - 举报达3次自动下架，触发人工复核
     - 恶意举报反向惩罚
   
   - **评价反馈**
     - 差评率 > 30%的设备主自动降级
     - 好评率 > 90%的设备主自动升级

4. **AI辅助审核**
   
   - **智能评分系统**
     - 图片质量评分（清晰度、相关性、重复度）
     - 文本质量评分（完整度、可读性、真实性）
     - 综合风险评分（0-100分）
   
   - **审核策略**
     - 风险评分 < 20：自动通过
     - 风险评分 20-60：机器预审 + 人工快速确认
     - 风险评分 > 60：人工详细审核

5. **审核流程优化**
   
   - **分级审核**
     - 低风险内容：自动通过（占80%）
     - 中风险内容：人工快速确认（占15%）
     - 高风险内容：人工详细审核（占5%）
   
   - **批量处理工具**
     - 相似内容批量审核
     - 同一用户内容批量操作
     - 一键通过/拒绝
     - 快捷拒绝理由模板
   
   - **审核历史**
     - 审核记录追溯
     - 审核员绩效统计
     - 审核规则动态调整

6. **规则引擎**
   
   - 配置化审核规则（无需改代码）
   - 动态调整阈值（根据运营数据）
   - 白名单/黑名单管理
   - A/B测试审核策略

**预期效果**：
- 新用户人工审核率：20%
- 普通用户人工审核率：5%
- 优质用户人工审核率：0%
- 整体人工审核率：< 10%
- 审核效率提升：10倍以上

**其他管理功能**：
    
- 增值排名付费管理
    
- 数据统计：访问量、点击量、付费统计、审核效率统计
    

### 3.4 增值功能（付费排名）

**排名档位（3档固定价格）**

1. **基础展示**：0元/月（免费）
   - 正常展示，无特殊标识
   - 按发布时间倒序排列

2. **推荐位**：100元/月
   - 标记"推荐"标识（橙色角标）
   - 优先于基础展示
   - 同档位内按付费时间倒序

3. **置顶位**：200元/月
   - 标记"置顶"标识（红色角标）
   - 最高优先级展示
   - 同档位内按付费时间倒序

**排序规则（同一区域内）**

排序优先级：
1. 置顶位（200元）→ 按付费时间倒序
2. 推荐位（100元）→ 按付费时间倒序
3. 基础展示（0元）→ 按发布时间倒序

**区域隔离策略**

- **省级推广**：全省范围展示（适合跨市服务）
- **市级推广**：全市范围展示（适合跨县服务）
- **县级推广**：全县范围展示（适合本地服务）

用户可选择推广区域范围，价格相同。

**技术实现**

排序字段计算：
```
排序分数 = 档位权重 + 时间戳
- 置顶位权重：1000
- 推荐位权重：500
- 基础展示权重：0
```

**防滥用机制**

- 同档位内按时间排序（避免价格战）
- 每个设备限购1个档位（不可重复购买）
- 付费到期自动降级为基础展示
- 到期前3天短信/站内信提醒续费

**付费管理**

- 月度自动续费（可关闭）
- 支持随时升级/降级
- 降级立即生效，退还剩余天数费用
- 升级补差价，立即生效

**展示位置**

- 首页：置顶位 > 推荐位 > 基础展示
- 搜索结果页：同上
- 分类页：同上
- 地区页：同上
    

### 3.5 SEO优化策略（搜索引擎自然流量）

**目标**：每个设备页面独立优化，最大化搜索引擎自然流量获取

**页面级SEO要求**：

1. **URL结构优化**
   - 语义化URL：`/{省}/{市}/{县}/{一级分类}/{二级分类}/{设备ID}-{型号}`
   - 示例：`/anhui/chuzhou/laian/agricultural/harvester/12345-kubota-688`
   - 全拼音或英文，避免中文编码

2. **页面标题（Title）**
   - 格式：`{型号} {机械类型} - {县}{市} - {价格/天} - 平台名`
   - 示例：`久保田688收割机 - 来安县滁州市 - 500元/天 - 重机通`
   - 长度：50-60字符，包含核心关键词

3. **Meta描述（Description）**
   - 格式：`{地区}{机械类型}{型号}，{作业能力}，{可租时间}，联系电话{部分号码}，{评分}星好评`
   - 示例：`滁州市来安县久保田688收割机，日作业50亩，5-10月可租，联系138****1234，4.8星好评`
   - 长度：120-160字符，突出核心信息

4. **页面标题标签（H1-H3）**
   - H1：`{型号} {机械类型} - {县}{市}`（唯一）
   - H2：设备参数、联系方式、用户评价、相关推荐
   - H3：具体参数项、评价详情

5. **结构化数据（Schema.org）**
   - Product类型：价格、评分、库存状态
   - LocalBusiness类型：地址、联系方式、营业时间
   - BreadcrumbList：面包屑导航
   - AggregateRating：聚合评分

6. **图片优化**
   - 文件名：`{机械类型}-{型号}-{序号}.jpg`
   - Alt属性：`{地区}{机械类型}{型号}实拍图{序号}`
   - 压缩优化：WebP格式，< 200KB
   - 懒加载：首屏外图片延迟加载

7. **内容丰富度**
   - 设备详细参数（表格形式）
   - 作业能力说明（文字描述）
   - 适用场景（关键词密度）
   - 用户评价（UGC内容）
   - 常见问题（FAQ结构化数据）

8. **内链优化**
   - 相关设备推荐（同类型、同地区）
   - 面包屑导航（省>市>县>分类>设备）
   - 上级分类页链接
   - 热门设备链接

9. **关键词布局**
   - 核心关键词：地区+机械类型+型号
   - 长尾关键词：地区+机械类型+租赁/出租/价格
   - 语义关键词：作业能力、适用场景、服务范围
   - 密度控制：2-3%，自然分布

10. **页面性能**
    - 首屏加载：< 2秒
    - LCP（最大内容绘制）：< 2.5秒
    - FID（首次输入延迟）：< 100ms
    - CLS（累积布局偏移）：< 0.1

11. **移动端优化**
    - 响应式设计，移动优先
    - 字体大小：≥ 16px
    - 按钮间距：≥ 48px
    - 视口适配：viewport meta标签

12. **更新频率**
    - 设备状态实时更新
    - 评价动态追加
    - 访问量定期更新
    - 最后更新时间显示

**列表页SEO要求**：

1. **分类页优化**
   - URL：`/{省}/{市}/{一级分类}/{二级分类}`
   - Title：`{市}{县}{二级分类}{一级分类}租赁信息 - 平台名`
   - 分页：rel="prev/next"标签

2. **地区页优化**
   - URL：`/{省}/{市}/{县}`
   - Title：`{县}{市}重型机械租赁信息大全 - 平台名`
   - 聚合内容：该地区所有机械

3. **Sitemap生成**
   - 自动生成XML sitemap
   - 每日更新，提交搜索引擎
   - 优先级：设备页0.8，分类页0.6，地区页0.7

4. **Robots.txt配置**
   - 允许抓取：设备页、分类页、地区页
   - 禁止抓取：用户中心、后台管理、API接口

**技术实现**：

- 服务端渲染（SSR）或静态生成（SSG）
- 预渲染关键页面
- CDN加速静态资源
- 自动生成sitemap和robots.txt
- 结构化数据自动注入
- 图片自动压缩和格式转换
    
- 排序：距离、热度、推荐位
    
- 地图展示（可选）

### 3.6 用户注册/登录

**注册方式**

1. **手机号注册**（主要方式）
   - 输入手机号
   - 发送短信验证码（60秒间隔）
   - 输入验证码验证
   - **强制设置密码**（6-20位，必须包含字母和数字）
   - 完成注册，自动登录

2. **微信快捷登录**（可选）
   - 微信授权登录
   - 首次需绑定手机号 + 设置密码

**登录方式**

1. **密码登录**（主要方式，节省短信费用）
   - 输入手机号 + 密码
   - 图形验证码（连续失败3次后显示）
   - 登录成功

2. **验证码登录**（备用方式）
   - 输入手机号
   - 发送短信验证码
   - 输入验证码登录
   - 提示：建议使用密码登录

**找回密码**

- 输入手机号
- 发送短信验证码
- 输入验证码验证
- 设置新密码
- 自动登录

**安全机制**

- 同一手机号每日验证码限制：10次
- 同一IP每日验证码限制：50次
- 密码登录失败5次锁定30分钟
- 异地登录提醒
- 密码强度检测（弱密码提示修改）

### 3.7 用户个人中心

**我的设备**

- **已发布**：正常展示中的设备
- **审核中**：待审核/审核中的设备
- **已拒绝**：审核未通过，显示拒绝原因
- **已下架**：用户主动下架/违规下架

每个设备显示：
- 基本信息（图片、标题、价格）
- 状态标识
- 浏览量/联系次数
- 操作按钮（编辑/删除/上架/下架/续费）

**我的收藏**

- 收藏的设备列表
- 显示设备状态（在线/已下架）
- 一键取消收藏

**浏览历史**

- 最近浏览的30条设备
- 按时间倒序
- 一键清空

### 3.8 设备发布流程

**发布入口**

**发布表单**

**必填字段**：

- 详细地址

**选填字段**：
- 设备描述（≥50字）

**草稿保存**

- 手动保存草稿
- 草稿有效期：7天

**发布后状态**

- 新用户：待审核（1-24小时）
- 普通用户：待审核（自动审核，5分钟内）

- 认证用户：直接发布

### 3.9 设备编辑/删除

**编辑权限**

- 仅设备发布者本人
- 管理员可编辑所有设备

**编辑规则**

- 基础信息修改：无需重新审核
- 图片/描述修改：需重新审核（普通用户）
- 价格/联系方式修改：无需审核

**删除机制**

- 删除前二次确认
- 删除后数据保留30天（可恢复）
- 30天后物理删除

**下架/上架**

- 用户可主动下架（临时不可用）
- 下架后不展示，可随时上架
- 上架后立即生效（无需审核）

### 3.10 联系方式查看

**查看权限**

- 必须登录后才能查看
- 未登录显示"登录后查看"

**查看流程**

1. 用户点击"查看手机号"或"查看微信"
2. 前端发送请求到后台
3. 后台验证权限和限制
4. 记录查看行为
5. 返回联系方式
6. 前端展示

**防刷机制**

- 同一用户查看同一设备：不限次数（已查看过直接显示）
- 同一用户每日查看不同设备：限50次
- 同一IP每日查看：限200次
- 异常行为（短时间大量查看）：临时封禁

**通知机制**

- 设备主收到站内信通知
- 通知内容：谁在什么时间查看了联系方式
- 可关闭通知

### 3.11 评价系统

**评价权限**

- 必须查看过联系方式的用户才能评价
- 每个用户对同一设备只能评价1次
- 评价后可修改（7天内）

**评价内容**

- 星级评分（1-5星）
- 文字评价（10-500字）
- 图片上传（0-3张，可选）
- 评价标签（服务好/价格合理/设备新/响应快等）

**评价审核**

- 自动审核：敏感词过滤
- 人工审核：恶意评价、无关内容
- 审核通过后展示

**评价展示**

- 设备详情页展示所有评价
- 按时间倒序
- 显示评价者昵称（脱敏）
- 显示评价时间
- 显示星级和内容

**恶意评价处理**

- 用户可举报恶意评价
- 举报达3次触发人工复核
- 确认恶意后删除评价，惩罚评价者

**退款规则**

### 3.13 通知系统

**通知类型**

### 3.16 异常处理

**设备下架规则**

1. **用户主动下架**
   - 随时可下架
   - 下架后不展示
   - 可随时上架

2. **违规下架**
   - 虚假信息：下架 + 警告
   - 重复发布：下架 + 降级
   - 违法内容：下架 + 封号

3. **付费到期**
   - 推荐/置顶到期自动降为基础展示
   - 不影响设备展示

4. **长期未更新**
   - 180天未更新自动下架
   - 短信提醒用户更新

**账号封禁规则**

1. **临时封禁**（1-7天）
   - 轻微违规3次
   - 恶意评价
   - 频繁举报

2. **永久封禁**
   - 发布违法内容
   - 严重虚假信息
   - 恶意刷单

**申诉流程**

1. 用户提交申诉
2. 上传证明材料
3. 人工审核（3个工作日）
4. 申诉结果通知
5. 通过后解除处罚

**数据恢复**

- 删除的设备30天内可恢复
- 封禁的账号申诉通过后恢复
- 误操作可联系客服恢复

## 六、盈利模式（免费基础 + 月度增值）

### 6.1 免费基础展示

- 所有机械信息免费展示
    
- 无门槛 → 快速吸引千万台机械加入
    

### 6.2 增值排名付费

- 排名越高 → 曝光越多 → 更多潜在客户
    
- 按月计费：100 元/月起，可阶梯
    
- 用户自主选择 → 不影响免费基础展示
    

### 6.3 收入预估（保守模型）

假设市场规模 1,000 万台机械，10%选择增值服务：

|参数|数值|
|---|---|
|总机械量|1,000 万台|
|选择付费比例|10% → 100 万台|
|月度付费|100 元/月起|
|年收入|100 万台 × 100 元 × 12 月 = 12 亿人民币/年|

> 与按年收费相比，月度付费模式收入更高且现金流稳定

### 3.19 距离计算

**地理位置获取**

- 用户首次访问请求位置权限
- 浏览器Geolocation API获取经纬度
- 拒绝授权则使用IP定位（精确到市）
- 位置缓存7天

**距离计算规则**

- 用户位置 ↔ 设备位置（县级中心点）
- 计算直线距离（Haversine公式）
- 显示规则：
  - < 1km：显示米（如：800米）
  - 1-100km：显示公里（如：15公里）
  - > 100km：显示公里（如：150公里）

**排序规则**

- 按距离从近到远排序
- 同距离按发布时间倒序

**地图展示（可选）**

- 列表页右上角"地图模式"切换
- 地图显示设备位置标记（县级）
- 点击标记显示设备简要信息
- 点击跳转详情页
