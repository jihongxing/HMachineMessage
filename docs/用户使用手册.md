## 一、快速开始

重型机械信息中介平台是一个专业的重型机械信息展示和撮合服务平台，为机械所有者和需求方提供便捷的信息发布和查询服务。

**平台特点**：
- 免费发布机械信息
- 精准搜索和筛选

- 安全可靠

**发布方**：
1. 注册/登录账号
2. 发布机械信息
3. 等待审核通过

2. 搜索和筛选机械

5. 联系机械所有者

## 二、注册与登录

### 2.1 手机号注册

**步骤**：
1. 点击"注册"按钮
2. 输入手机号
3. 点击"获取验证码"
4. 输入收到的验证码
5. 设置密码（6-20位，包含字母和数字）
6. 点击"注册"完成

**注意事项**：
- 手机号必须真实有效
- 密码需包含字母和数字
- 验证码60秒后可重新获取

### 2.2 微信快捷登录

**步骤**：
1. 点击"微信登录"按钮
2. 扫描二维码授权
3. 首次需绑定手机号
4. 设置密码
5. 完成登录

### 2.3 密码登录

**步骤**：
1. 点击"登录"按钮
2. 输入手机号和密码
3. 点击"登录"

**提示**：建议使用密码登录，节省短信费用。

### 2.4 验证码登录

**步骤**：
1. 点击"验证码登录"
2. 输入手机号
3. 获取验证码
4. 输入验证码
5. 点击"登录"

### 2.5 找回密码

**步骤**：
1. 点击"忘记密码"
2. 输入手机号
3. 获取验证码
4. 输入验证码
5. 设置新密码
6. 完成重置

## 三、发布机械信息

### 3.1 发布入口

- 首页"发布设备"按钮
- 个人中心"发布设备"
- 移动端底部导航栏

### 3.2 填写信息

**必填字段**：
- **设备分类**：选择一级和二级分类
- **设备型号**：如"久保田688"
- **所在地区**：选择省、市、县
- **详细地址**：具体位置
- **价格**：输入价格和单位（元/天 或 元/小时）
- **联系电话**：您的手机号
- **设备图片**：上传3-9张图片

**选填字段**：
- **设备描述**：详细介绍（建议≥50字）
- **作业能力**：如"50亩/天"
- **可租时间**：开始和结束日期
- **微信号**：方便联系
- **适用场景**：如"水稻收割"

### 3.3 上传图片

**要求**：
- 数量：3-9张
- 格式：JPG、PNG、WebP
- 单张大小：< 5MB
- 总大小：< 30MB

**建议**：
- 第一张为主图（清晰、正面）
- 包含设备全景、细节、铭牌
- 图片清晰、光线充足
- 避免水印和无关内容

### 3.4 提交审核

**审核时间**：
- 新用户：1-24小时（人工审核）
- 普通用户：5分钟内（自动审核）
- 优质用户：免审核，直接发布
- 认证用户：免审核，直接发布

**审核标准**：
- 信息真实完整
- 图片清晰相关
- 内容合法合规
- 无重复发布

### 3.5 审核结果

**通过**：
- 收到通知
- 信息正常展示
- 可查看浏览数据

**拒绝**：
- 收到通知和原因
- 可修改后重新提交
- 3次拒绝需人工审核

## 四、搜索机械

### 4.1 搜索方式

**关键词搜索**：
- 输入型号、品牌、类型
- 如"久保田"、"收割机"、"688"

- 选择一级分类（农业/工程/运输/其他）

**价格筛选**：
- 选择价格区间
- 如"0-500元/天"

**推荐位筛选**：
- 基础展示
- 推荐位
- 置顶位

### 4.2 排序方式

- **最新发布**：按发布时间倒序
- **价格从低到高**：按价格升序
- **价格从高到低**：按价格降序

**开启方式**：

**功能**：

## 七、增值服务

### 7.1 服务类型

**基础展示**（0元/月）：
- 免费
- 正常展示
- 按发布时间排序

**推荐位**（100元/月）：
- 橙色"推荐"标识
- 优先于基础展示
- 按付费时间排序

**置顶位**（200元/月）：
- 红色"置顶"标识
- 最高优先级
- 按付费时间排序

### 7.2 购买流程

**步骤**：
1. 进入"我的设备"
2. 选择要推广的设备
3. 点击"购买增值服务"
4. 选择档位（推荐/置顶）
5. 选择推广区域（省/市/县）
6. 选择时长（1-12个月）
7. 确认订单
8. 选择支付方式
9. 完成支付
10. 自动生效

### 7.3 支付方式

- 微信支付
- 支付宝支付
- 余额支付

### 7.4 自动续费

**开启方式**：
- 购买时勾选"自动续费"
- 个人中心"订单管理"开启

**规则**：
- 到期前3天自动扣费
- 余额不足短信提醒
- 可随时关闭

### 7.5 升级/降级

**升级**：
- 补差价，立即生效
- 如：推荐位 → 置顶位，补100元

**降级**：
- 立即生效，退还剩余天数费用
- 如：置顶位 → 推荐位，退还剩余天数的100元

### 7.6 退款规则

- 付费后24小时内可申请全额退款
- 超过24小时按剩余天数比例退款
- 退款到原支付方式
- 3-7个工作日到账

### 7.7 发票开具

**申请方式**：
- 个人中心"订单管理"
- 点击"申请发票"

**发票信息**：
- 发票类型：增值税普通发票
- 发票内容：信息服务费
- 开票信息：个人/企业

## 八、常见问题

### 8.1 注册登录

**Q：收不到验证码怎么办？**
A：
1. 检查手机号是否正确
2. 检查短信是否被拦截
3. 等待60秒后重新获取
4. 联系客服处理

**Q：忘记密码怎么办？**
A：点击"忘记密码"，通过手机验证码重置密码。

**Q：可以更换手机号吗？**
A：可以，在个人中心"个人信息"中更换，需验证新旧手机号。

### 8.2 发布机械

**Q：发布需要收费吗？**
A：发布机械信息完全免费，增值排名服务需付费。

**Q：审核需要多久？**
A：
- 新用户：1-24小时
- 普通用户：5分钟内
- 优质用户：免审核

**Q：审核不通过怎么办？**
A：查看拒绝原因，修改后重新提交。

**Q：可以发布多少条信息？**
A：没有数量限制，但不得重复发布相同信息。

**Q：如何提高审核通过率？**
A：
1. 信息真实完整
2. 图片清晰相关
3. 描述详细准确
4. 避免敏感词汇

### 8.3 搜索查看

**Q：如何查看联系方式？**
A：登录后点击"查看手机号"或"查看微信"按钮。

**Q：查看联系方式有限制吗？**
A：有限额，新用户/普通用户50次/天，优质用户100次/天，认证用户200次/天。

**Q：如何计算距离？**
A：需授权位置权限，系统自动计算您与设备的直线距离。

**Q：为什么有些设备没有联系方式？**
A：可能是发布者未填写或已下架。

### 8.4 增值服务

**Q：增值服务有什么用？**
A：提高设备曝光率，获得更多联系机会。

**Q：如何选择档位？**
A：
- 基础展示：免费，适合试用
- 推荐位：100元/月，适合一般推广
- 置顶位：200元/月，适合重点推广

**Q：可以退款吗？**
A：付费后24小时内可全额退款，超过24小时按剩余天数比例退款。

**Q：如何开发票？**
A：在个人中心"订单管理"中申请，支持电子发票。

### 8.5 账号安全

**Q：如何保护账号安全？**
A：
1. 设置复杂密码
2. 不要泄露密码
3. 定期修改密码
4. 不要在公共设备登录

**Q：账号被盗怎么办？**
A：立即联系客服，冻结账号，修改密码。

**Q：如何注销账号？**
A：在个人中心"账户管理"中申请注销，需结清所有费用。

### 8.6 其他问题

**Q：平台收取交易佣金吗？**
A：不收取，平台仅提供信息展示服务，不参与交易。

**Q：如何投诉举报？**
A：在设备详情页点击"举报"按钮，选择举报原因。

## 九、用户等级说明

### 9.1 等级体系

**新用户**（0级）：
- 注册后默认等级
- 首次发布需人工审核
- 查看联系方式限50次/天

**普通用户**（1级）：
- 发布5条通过后自动升级
- 自动审核 + 5%抽查
- 查看联系方式限50次/天

**优质用户**（2级）：
- 发布20条通过 + 0违规后自动升级
- 免审核直接发布
- 查看联系方式限100次/天

**认证用户**（3级）：
- 完成实名认证或企业认证
- 免审核 + 优先展示
- 查看联系方式限200次/天
- 可申请提现

### 9.2 如何升级

**升级到普通用户**：
- 发布5条信息并通过审核

**升级到优质用户**：
- 发布20条信息并通过审核
- 无违规记录

**升级到认证用户**：
- 完成实名认证（个人）
- 完成企业认证（企业）
